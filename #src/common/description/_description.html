Описание

Лазерный маркер TOR LT

Cтанок для гравировки и маркировки по различным материалам таким как, металлы, пластики, кожа, ткани с неорганическими наполнителями и красителями. Модель TOR LT выполнена в настольном исполнении. Регулировка по высоте осуществляется путем поднятия и опускания координатного столика, регулируемого по оси Z в ручную. Данный маркер подойдет для работы с небольшими заготовками. На модель TOR LT можно установить рабочее поле 100х100 мм или 200х200 мм, станок отлично подойдет тем, кто занимается ювелирной или рекламной деятельностью.

Данное оборудование отлично подходит для прецизионной маркировки и гравировки. Маркер выполнен в настольном варианте. Лазерная голова регулируется по высоте. Позволяет работать с крупными и мелкими деталями, также на конвейере. Легко встраивается в линию.

Рабочее поле

Комплектация TOR LT включает линзу с полем 100x100 мм.
